# 📦 Logistics Delivery Analytics

## 📌 Project Overview
End-to-end logistics analytics project analyzing 500+ shipments 
across major Indian cities — Delhi, Mumbai, Chennai, Bangalore, 
and Kolkata.

---

## ❓ Business Questions Answered
- What is the on-time delivery rate across cities and routes?
- Which weather, traffic, and road conditions cause the most delays?
- Which vehicle types and drivers perform best?
- How do Corporate, Retail, and Wholesale customers differ in fulfillment?
- Which product categories (Electronics, Furniture, Food etc.) face most delays?

---

## 📊 Dataset Details
| Column | Description |
|---|---|
| Shipment ID | Unique delivery identifier |
| Origin & Destination | City-to-city routes |
| Vehicle Type | Truck, Van, Bike etc. |
| Delivery Status | On-time / Delayed |
| Delay Days | Number of days delayed |
| Weather / Traffic / Road | External condition factors |
| Customer Type | Corporate / Retail / Wholesale |
| Product Category | Electronics, Furniture, Food etc. |
| Fuel Cost | Cost per shipment |
| Driver Rating | Performance score |

---

## 🔍 Key Findings
- Identified high-risk routes and condition combinations causing maximum delays
- Storm weather and high-traffic routes showed highest delay rates
- Corporate customers had stricter delivery expectations vs Retail/Wholesale
- Built Excel dashboards with pivot tables to visualize KPIs
